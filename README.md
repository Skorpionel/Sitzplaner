# Sitzplaner

Ein simpler Sitzplan Editor für den Klassenraum. Tische kann man per Drag & Drop platzieren, Sitznummern kann man auch mischen, und voila.

## Nutzung

Bei den Releases ([Link hier](https://github.com/Skorpionel/Sitzplaner/releases)) einfach Sitzplaner_Release_V.x.x.zip herunterladen und extrahieren. Danach einfach die Sitzplaner.exe Datei ausführen und benutzen. Dabei ist kein Python nötig, da alle Dateien schon fertig als `.exe` Dateien vorgestellt sind.

## Fähigkeiten
Tische hinzufügen, verschieben, drehen und löschen. Doppelklick auf einen Tisch zum Bearbeiten der Sitznummern (dies kann ein bisschen Fehlerhaft werden). Der "Sitze mischen" Knopf verteilt alle Nummern in den Tischen zufällig neu. Updates werden normalerweise automatisch erkannt und eingewendet (nach einer Nachfrage).
